# feedly server

## how to get tokens
`https://feedly.com/v3/auth/dev`ここにアクセスしてログインすると，登録されてるアドレスに対してuser id と access tokenが取得できるリンクが送られてくる．
access tokenの有効期限は一ヶ月だから様子見てみて良ければproを契約する．
